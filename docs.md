# Документация

---

## Роли

---

### Администратор (ADMIN)

Сотрудник, чья задача — управлять пользователями информационной системы

#### Ответственность

Управление пользователями:
- Регистрация новых пользователей
- Изменение существующих пользователей
- Управление статусом пользователей
- Фильтрация (поиск) пользователей

---

### Оператор (OPERATOR)

Сотрудник, чьи задачи:
1. Принимать обращения граждан
2. Записывать новых граждан в базу

#### Ответственность

Управление гражданами (возможно уберу):
- Создание новых записей о гражданах, которых ещё нет в базе данных
- Изменение существующих граждан
- Удаление записей о гражданах (если у них нет обращений)
- Фильтрация (поиск) граждан

Управление обращениями:
- Создание обращения
- Просмотр комментариев и статуса обращения (НОВОЕ, В ОБРАБОТКЕ, ГОТОВО, ОТКЛОНЕНО)
- Фильтрация (поиск) обращений

---

### Сотрудник (EMPLOYEE)

Сотрудник, чья задача — обработка обращений граждан

#### Ответственность

Обработка обращений:
- Изменение статуса обращений
- Написание комментариев к обращению
